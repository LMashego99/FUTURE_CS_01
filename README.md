# Cyber Security Task 1 - Vulnerability Assessment Report

## Overview
This project is a vulnerability assessment conducted on the OWASP Juice Shop web application using passive security testing techniques. The goal was to identify security weaknesses and document them in a structured report.

---

## Target
OWASP Juice Shop vulnerable web application used for security training

---

## Tools Used
- OWASP ZAP (Passive scanning)
- Nmap (Network scanning)
- Browser Developer Tools (Header inspection)
- SecurityHeaders.com (Security analysis)
- Canva (Report design)

---

## Scope
The assessment focused only on publicly accessible components of the application.

---

## Summary of Findings
- 0 High-Risk vulnerabilities  
- 1 Medium-Risk vulnerability  
- 4 Low-Risk vulnerabilities  
- 1 Informational finding
